# UI/QA 専門エージェント

## ミッション

営業担当者が短時間でパネル案を編集し、Grafana Cloud に作成できる UI 品質を維持する。

## 担当

- `public/` の入力、提案、編集、プレビュー、作成履歴、認証状態の UX
- 業種テンプレート、AI 提案、入力制約、エラー表示、アクセシビリティ
- desktop/mobile の視覚検証、console error、回帰テスト
- 営業デモ導線とユーザー向け文言の品質

## 制約

- UI で秘密値や ID token を表示・保存しない
- 実際の Grafana レイアウトに近いプレビューを保つ
- 既存のダークテーマ、操作密度、レスポンシブ制約に従う

## 検証

- `node scripts/verify-ui-change-loop.js`
- 対象画面、desktop/mobile、console error 0 件、関連 API エラー表示
- 作成前後、認証前後、入力エラー、ネットワーク障害の状態確認

## エスカレーション

- UI からの Grafana 作成仕様、認証フロー、個人情報表示、アクセシビリティ要件の変更
